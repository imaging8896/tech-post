# 0 手動 - 用 Github Copilot 打造服務

**Issue #4**
**Author:** imaging8896
**Created:** 2026-02-01T09:20:09Z

## Description

**Overview:**
不需要 IDE 不需要寫 code 也不需要 git commit and push，你只需要把需求定義好，服務就完成。

**Prerequisites:**
1. 擁有一個 Github 帳號
2. 訂閱 Github Copilot

**Step-by-Step Guide:**
1. 登入 Github 首頁後，選 **Repositories**，點 **New**
2. 填入專案名稱（最好符合你想建立的服務）
3. 在 **Jumpstart your project with Copilot (optional)** 的 ** Prompt** 內 **詳細** 填入你想要服務的功能（以我的範例是『我希望能夠把我學習到的知識做成文章發佈到 Linkedin』）
4. 完成輸入後點選 **Create repository**
5. 會需要一點時間，完成後，Copilot 會告知完成的內容，請讀一下看看結果
6. 確定大致沒問題，點選右下角 **View pull request**
7. 到達了修改確認頁面（**Pull requests**），確認無誤後點最下方的 **Merge**
8. 你的服務應該就完成了
9. 可能會有後續的服務設定，請參照剛剛新增的檔案，Copilot 通常會生成說明文件，請按照文件進行設定

**Common Pitfalls:**
1. 不會一次就完美建成服務，畢竟一開始你想要的功能可能無法達成或是有限制，需要繼續跟 Copilot 來回對話調整
2. 可能需要設定 Github（下一篇）
3. 可能需要注意安全性（下一篇）

**Conclusion:**

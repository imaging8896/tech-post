# AI時代，研究方法或做事方法更為重要

**Issue #22**
**Author:** imaging8896
**Created:** 2026-05-02T04:49:56Z

## Description

AI可以查資料，可以寫程式，但是一旦叫他完成一個複雜任務，他可能會往錯誤的方向進行，或是叫他進行研究，那可能得出錯誤的研究結論；在此我要謝謝我研究所教授 中央大學梁德容教授，他當時傳授我的研究方法論，至今雖然不進行研究了，但也成為我做事方法論的一部分。

研究方法
1. 提出假說，假說需要有「直覺」可能成立的理由，理由可以透過簡單的邏輯論證確立
2. 實驗和資料證明假說
3. 結論

我將以上設定成agent 的 SOUL.md，而他每天每小時都可以進行研究，並把我的小專案拉到我沒想到的高度。

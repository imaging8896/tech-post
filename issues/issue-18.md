# 使用AI萬事都可以創業的時代來了

**Issue #18**
**Author:** imaging8896
**Created:** 2026-03-16T14:50:35Z

## Description

幾天前才橫空出世的工具https://github.com/paperclipai/paperclip，讓你也可以體驗創業的感覺，需要的東西有 claude subscription(or API token)，網頁起來之後定義公司目標並且 hire 一個 CEO，公司開始運作，CEO其實就是一個 AI agent，他會決定這個公司未達成目標需要做什麼事情，可能是招聘工程師寫程式，於是公司多了個工程師，當然，他也是AI agent，於是一群 AI agents 在運營公司，幫你完成你想要的事情

至於能做什麼，靜待下篇吧

要注意，使用API token，會耗費一些錢，但是只要幾百幾千台幣，就能開始一間公司，這是以前不會發生的事情
